# Custom Web Application Firewall (WAF)

## Overview
This project implements a basic Web Application Firewall using Python and Flask to inspect HTTP requests and block common web-based attacks.

## Objective
- Detect malicious user input
- Block SQL Injection, XSS, and Command Injection
- Log blocked requests for monitoring

## Tools Used
- Python
- Flask
- HTML
- Windows OS

## How to Run
pip install flask
python waf.py

Open browser:
http://127.0.0.1:5000

## Security Concepts Demonstrated
- Input validation
- Application-layer security
- Logging and monitoring

## Disclaimer
This project was developed in a controlled lab environment for educational purposes only.
